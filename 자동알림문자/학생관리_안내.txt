연락처 수정, 신규 학생 등록, 학생 삭제 기능이 추가되었습니다!


🔧 관리 기능 (3종)
==================

1. **📞 연락처수정** - 학생 전화번호 변경
2. **➕ 학생등록** - 신규 학생 추가
3. **🗑️ 학생삭제** - 학생 정보 삭제

📞 연락처 수정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사용 시나리오:
• 학부모가 전화번호 변경
• 잘못 입력된 번호 수정
• 새 연락처로 업데이트

사용 방법:
1. 학생 선택
2. [📞 연락처수정] 버튼 클릭
3. 현재 연락처 확인
4. 새 연락처 입력
   - 010-1234-5678 형식
   - 01012345678 형식
   - 둘 다 가능!
5. [확인] 클릭
6. 수정 완료!

예시:
┌─────────────────────────────┐
│ 연락처 수정                  │
├─────────────────────────────┤
│ 홍길동님의 새 연락처:       │
│                             │
│ 현재: 010-1234-5678         │
│ 형식: 010-1234-5678         │
│                             │
│ [입력창]                    │
│                             │
│ [확인]        [취소]        │
└─────────────────────────────┘

💡 Tip:
• 기존 번호가 미리 표시됨
• 일부만 수정 가능
• 즉시 엑셀에 반영


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
➕ 학생 등록
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사용 시나리오:
• 신규 학생 입학
• 학기 시작
• 추가 등록

사용 방법:
1. [➕ 학생등록] 버튼 클릭
2. Step 1/3: 이름 입력
3. Step 2/3: 연락처 입력
4. Step 3/3: 납입일 입력 (선택사항)
   - 입력하면 납입완료
   - 빈칸이면 미납
5. [확인] 클릭
6. 등록 완료!

입력 예시:

Step 1/3 - 이름
┌─────────────────────────────┐
│ 학생 등록 - 1/3              │
├─────────────────────────────┤
│ 학생 이름을 입력하세요:      │
│                             │
│ [홍길동]                    │
│                             │
│ [확인]        [취소]        │
└─────────────────────────────┘

Step 2/3 - 연락처
┌─────────────────────────────┐
│ 학생 등록 - 2/3              │
├─────────────────────────────┤
│ 홍길동님의 연락처:           │
│ (예: 010-1234-5678)         │
│                             │
│ [010-1234-5678]             │
│                             │
│ [확인]        [취소]        │
└─────────────────────────────┘

Step 3/3 - 납입일
┌─────────────────────────────┐
│ 학생 등록 - 3/3              │
├─────────────────────────────┤
│ 홍길동님의 원비 납입일:      │
│ (선택사항, 빈칸이면 미납)    │
│ (예: 2024-01-15)            │
│                             │
│ [2024-01-15]                │
│                             │
│ [확인]        [취소]        │

💡 Tip:
• 납입일은 선택사항 (나중에 등록 가능)
• 자동으로 하원 상태(0)로 등록
• 엑셀 마지막 행에 추가
• 즉시 목록에 표시

🗑️ 학생 삭제
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

사용 시나리오:
• 학생 퇴원
• 학기 종료
• 졸업
• 잘못 등록된 데이터 삭제

사용 방법:
1. 삭제할 학생 선택
2. [🗑️ 학생삭제] 버튼 클릭
3. 확인 팝업 (학생 정보 확인)
4. [예] 클릭
5. 삭제 완료!

확인 팝업:
┌─────────────────────────────┐
│ 학생 삭제                    │

│                             │
│ 연락처: 010-1234-5678        │
│ 상태: 등원중                 │
│                             │
│ ⚠️ 이 작업은 되돌릴 수 없습니다!│

│ [예]          [아니오]      │

• 삭제 후 복구 불가!

💡 활용 시나리오
===============

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
시나리오 1: 신규 학생 입학
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상황:
새 학생이 입학함

처리:
1. [➕ 학생등록] 클릭
2. 이름: 김민수
3. 연락처: 010-9876-5432
4. 납입일: (빈칸 - 아직 납입 안함)
5. 등록 완료!
6. 목록에 즉시 표시
7. 등원 시 [등원] 버튼으로 처리


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
시나리오 2: 전화번호 변경
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상황:
학부모 전화번호가 변경됨

처리:
1. 해당 학생 선택
2. [📞 연락처수정] 클릭
3. 새 번호 입력: 010-1111-2222
4. 수정 완료!
5. 다음 알림부터 새 번호로 발송


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
시나리오 3: 학생 퇴원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상황:
학생이 학원을 그만둠

처리:
1. 해당 학생 선택
2. [🗑️ 학생삭제] 클릭
3. 정보 확인
4. [예] 클릭
5. 삭제 완료!
6. 목록에서 제거됨


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
시나리오 4: 잘못 입력된 데이터 수정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상황:
이름은 맞는데 전화번호가 잘못됨

처리:
1. 해당 학생 선택
2. [📞 연락처수정] 클릭
3. 올바른 번호 입력
4. 완료!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
시나리오 5: 학기 시작 - 대량 등록
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

상황:
새 학기에 학생 여러 명 등록

방법 1 (GUI):
1. [➕ 학생등록] 반복 클릭
2. 한 명씩 등록
3. 정확하지만 시간 소요

방법 2 (엑셀 직접):
1. 엑셀 파일 열기
2. 학생 정보 일괄 입력
3. 저장
4. 프로그램에서 새로고침

💡 추천: 5명 이상이면 엑셀 직접 수정이 빠름

🔧 문제 해결
===========

Q: 버튼이 안 보여요
A:
  GUI 프로그램 재시작
  (최신 버전으로 업데이트됨)

Q: 연락처 수정이 안 돼요
A:
  ☑ 엑셀 파일이 열려있나요? → 닫기
  ☑ 번호 형식이 맞나요?
  ☑ 프로그램을 새로고침했나요?

Q: 학생 등록 후 안 보여요
A:
  [🔄 새로고침] 버튼 클릭

Q: 삭제했는데 다시 나타나요
A:
  ☑ 엑셀 파일이 저장되었나요?
  ☑ 다른 프로그램에서 엑셀을 열었나요?

Q: 실수로 삭제했어요
A:
  안타깝지만 복구 불가능 😢
  → 정기적으로 엑셀 파일 백업 필수!
