════════════════════════════════════════════════════════════
     알리고 템플릿 등록 방법
════════════════════════════════════════════════════════════

[STEP 1] 알리고 로그인
https://smartsms.aligo.in/

[STEP 2] 메뉴 이동
카카오 알림톡 → 템플릿 관리 → 템플릿 등록

[STEP 3] 기본 정보 입력
• 템플릿명: "학원_등하원_통합알림"
• 카테고리: 교육 > 학원/교습소
• 템플릿 유형: 정보성 메시지

[STEP 4] 템플릿 내용 입력
위의 템플릿 3번 내용을 그대로 복사 붙여넣기

[STEP 5] 변수 등록
#{학원명}, #{이름}, #{상태}, #{시각}

[STEP 6] 검수 신청
저장 후 검수 신청 버튼 클릭

[STEP 7] 승인 대기
보통 1~2 영업일 소요
(빠르면 몇 시간)

[STEP 8] 승인 완료 후
템플릿 코드 확인 → sms_config.json에 입력


════════════════════════════════════════════════════════════
     승인 잘 받는 팁
════════════════════════════════════════════════════════════

✅ DO (이렇게 하세요)
──────────────────
• 정보 전달 목적의 내용
• 명확한 변수명 사용 (#{변수명})
• 학원 이름, 학생 이름 등 필수 정보
• 자동 발송 안내 문구 포함
• 깔끔하고 간결한 문구


❌ DON'T (하지 마세요)
─────────────────────
• 광고성 문구 ("할인", "이벤트" 등)
• 부정확한 변수명
• 너무 긴 내용 (500자 제한)
• 이모지 과다 사용
• 외부 링크 (버튼은 가능)
