📊 엑셀 파일 구조
=================

파일명: 202511_자동알림.xlsx

┌────────────────────────────────────────────────────────┐
│  A열     │    B열        │  C열    │    D열          │
│ (이름)   │  (연락처)     │ (상태)  │ (원비납입일)    │
├──────────┼───────────────┼─────────┼─────────────────┤
│ 홍길동   │ 010-1234-5678 │   0     │ 2024-01-15      │
│ 김영희   │ 010-2345-6789 │   1     │                 │
│ 이철수   │ 010-3456-7890 │   0     │ 01/20           │
│ 박민수   │ 010-4567-8901 │   1     │ 2024-01-22      │
└──────────┴───────────────┴─────────┴─────────────────┘


📋 각 열 설명
============

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
A열 - 이름
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 학생 이름 입력
• 필수 항목 (이름이 없으면 해당 행은 무시됨)

예시:
  홍길동
  김영희
  이철수


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
B열 - 연락처
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 학부모 전화번호
• SMS/카카오톡 발송 대상

형식:
  ✓ 010-1234-5678 (하이픈 포함)
  ✓ 01012345678 (하이픈 없음)
  둘 다 가능!

💡 Tip:
  - 숫자로 저장되면 앞의 0이 사라질 수 있음
  - 셀 서식을 "텍스트"로 설정 권장
  - 또는 ' 을 앞에 붙여서 입력 ('01012345678)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
C열 - 등원/하원 상태
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 현재 등원/하원 상태
• 숫자로 입력

값:
  0 = 하원 상태
  1 = 등원중

동작:
  • 0 → 1로 변경: 등원 문자 발송
  • 1 → 0으로 변경: 하원 문자 발송

💡 Tip:
  - 프로그램에서 자동 변경됨
  - 수동으로 변경도 가능
  - 꼭 0 또는 1이어야 함


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
D열 - 원비 납입일 ⭐ NEW!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 원비 납입한 날짜
• 선택 항목 (없으면 미납)

형식:
  ✓ 2024-01-15
  ✓ 2024/01/15
  ✓ 01/15
  ✓ 1/15
  ✓ 어떤 형식이든 OK!

표시:
  • 날짜 있음 → "납입완료 (날짜)" 표시
  • 날짜 없음 → "미납" 표시

💡 Tip:
  - 프로그램에서 [납입등록] 버튼으로 입력
  - 또는 엑셀에서 직접 입력
  - 빈칸으로 두면 미납


✏️ 엑셀 파일 작성 방법
====================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 1: 기존 파일 수정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 202511_자동알림.xlsx 파일 열기
2. 학생 정보 수정/추가
3. 저장
4. 프로그램에서 자동 반영


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 2: 새로 만들기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 엑셀 새 파일 만들기

2. 1행(헤더) 작성:
   A1: 이름
   B1: 연락처
   C1: 상태
   D1: 납입일

3. 2행부터 학생 정보 입력:
   A2: 홍길동
   B2: 010-1234-5678
   C2: 0
   D2: 2024-01-15

4. 파일명으로 저장:
   202511_자동알림.xlsx


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 3: 샘플 복사
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 셀 서식 설정
==============

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
B열 (연락처) 서식
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제: 010-1234-5678 입력 시 숫자로 변환됨

해결 방법:

1. B열 전체 선택
2. 우클릭 → 셀 서식
3. "텍스트" 선택
4. 확인
5. 전화번호 입력

또는:

입력 시 앞에 ' 붙이기
  '010-1234-5678


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 숫자 형식으로 유지
• 0 또는 1만 입력

• 텍스트 또는 날짜 형식

⚠️ 주의사항
==========

1. 컬럼 순서 변경
   A, B, C, D 순서 유지!

2. 헤더 행 삭제
   1행은 헤더용으로 비워두기

3. 빈 행 삽입
   학생 사이에 빈 행 넣지 않기

   이름순, 납입순 등 정렬 가능

   미납 학생만 보기 등

4. 조건부 서식
   색상으로 구분하기


🔍 문제 해결
===========

Q: 학생이 목록에 안 나와요
A:
  ☑ A열(이름)에 값이 있나요?
  ☑ 2행부터 데이터가 있나요?
  ☑ 파일명이 정확한가요?
  ☑ 프로그램을 재시작했나요?

Q: 전화번호 앞에 0이 사라져요
A:
  ☑ B열 서식을 "텍스트"로 설정
  ☑ 또는 ' 을 앞에 붙여서 입력

Q: 상태 변경이 안 돼요
A:
  ☑ C열이 0 또는 1인가요?
  ☑ 숫자 형식인가요? (문자 아님)
  ☑ 엑셀 파일이 저장되었나요?

Q: 납입일이 표시 안 돼요
A:
  ☑ D열에 값이 있나요?
  ☑ 엑셀 파일이 저장되었나요?
  ☑ 프로그램을 새로고침했나요?

💡 Pro Tip
=========

1. 백업 만들기
   - 매주 또는 매월 파일 복사
   - 날짜를 파일명에 포함
   - 예: 202511_자동알림_백업_20240115.xlsx

2. 클라우드 저장
   - OneDrive, Google Drive 등에 저장
   - 자동 백업 및 어디서든 접근

3. 엑셀 팁
   - Ctrl+Home: 첫 셀로 이동
   - Ctrl+End: 마지막 셀로 이동
   - Ctrl+F: 학생 이름 검색

4. 필터 활용
   - 미납 학생만 보기
   - 등원중인 학생만 보기
   - D열 기준 정렬
