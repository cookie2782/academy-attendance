     원비 납입 관리 기능 안내
════════════════════════════════════════════════════════════

D열에 원비 납입일을 기록하고 관리하는 기능이 추가되었습니다!


📊 엑셀 파일 구조
=================

| A열 (이름) | B열 (연락처) | C열 (상태) | D열 (납입일) |
|-----------|-------------|-----------|-------------|
| 홍길동     | 010-1234-5678 | 0 | 2024-01-15 |
| 김영희     | 010-2345-6789 | 1 |            |
| 이철수     | 010-3456-7890 | 0 | 01/20      |

- **D열**: 원비 납입일
  - 날짜가 있으면: "납입완료" 표시
  - 날짜가 없으면: "미납" 표시


💡 납입 상태 표시
================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
GUI 프로그램 (등하원입력_GUI.bat)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

학생 목록에 "원비 납입" 컬럼이 추가됨:
┌────┬─────┬──────────┬────────┬──────────┐
│번호│이름 │ 연락처   │ 상태   │ 원비납입 │
├────┼─────┼──────────┼────────┼──────────┤
│ 1  │홍길동│010-1234  │✓ 등원중│ 납입완료 │
│ 2  │김영희│010-2345  │○ 하원  │   미납   │
└────┴─────┴──────────┴────────┴──────────┘

배경색으로 구분:
• 등원중 + 납입완료: 초록색
• 하원 + 납입완료: 연한 빨간색
• 등원중 + 미납: 노란색
• 하원 + 미납: 진한 빨간색


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
콘솔 프로그램 (등하원입력_콘솔.bat)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────┬────────┬──────────────┬───────┬──────────┐
│번호 │ 이름   │   연락처     │ 상태  │ 원비납입 │
├─────┼────────┼──────────────┼───────┼──────────┤
│ 1   │홍길동  │010-1234-5678 │✓등원중│ 납입완료 │
│ 2   │김영희  │010-2345-6789 │○하원  │  미납   │
└─────┴────────┴──────────────┴───────┴──────────┘


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
웹 앱 (웹앱실행.bat)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

각 학생 카드에 납입 상태 표시:

┌──────────────────────┐
│ 홍길동               │
│ 📱 010-1234-5678     │
│ ✓ 등원중             │
│ 💰 납입완료 2024-01-15│
│ [등원] [하원] [납입등록] │
└──────────────────────┘

• 납입완료: 초록색 배지
• 미납: 빨간색 배지


🔧 납입일 등록 방법
==================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 1: GUI 프로그램 (추천!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. [등하원입력_GUI.bat] 실행
2. 학생 선택
3. [💰 납입등록] 버튼 클릭
4. 납입일 입력 (예: 2024-01-15 또는 01/15)
5. [확인] 클릭
6. 완료!

💡 납입 정보 삭제:
  - [납입등록] 버튼 클릭
  - 빈칸으로 확인
  - "삭제하시겠습니까?" → 예


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 2: 웹 앱 (PC/모바일)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 웹 앱 접속
2. 학생 카드에서 [납입등록] 버튼 클릭
3. 팝업창에 납입일 입력
4. [확인] 클릭
5. 완료!

모바일에서도 동일하게 [납입] 버튼 터치


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
방법 3: 엑셀 직접 수정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 엑셀 파일 열기
2. D열에 날짜 입력
   - 2024-01-15
   - 01/15
   - 2024-1-15
   - 어떤 형식이든 OK!
3. 저장
4. 프로그램에서 자동 반영


📅 날짜 형식
===========

다양한 형식을 지원:

✓ 2024-01-15
✓ 2024/01/15
✓ 2024.01.15
✓ 01/15
✓ 1/15
✓ 2024-1-15
✓ 01-15

💡 어떤 형식이든 입력하신 그대로 저장됩니다!


💡 활용 예시
===========

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
월초 원비 납입 체크
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

매월 1일:
1. GUI 프로그램 실행
2. 목록에서 "미납" 학생 확인
3. 납입 받으면 즉시 [납입등록] 버튼으로 등록
4. 한눈에 납입 현황 파악!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
미납 학생 관리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 프로그램에서 "미납" 표시 확인
2. 해당 학생에게 연락
3. 납입 후 즉시 등록
4. 실시간 업데이트!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
모바일에서 현장 등록
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

학생이 현금 납입 시:
1. 스마트폰으로 웹 앱 접속
2. 해당 학생 카드에서 [납입] 터치
3. 오늘 날짜 입력
4. 즉시 처리 완료!


🎨 색상 구분
===========

GUI 프로그램에서 배경색으로 한눈에 파악:

┌────────────────────────────────────┐
│ 등원중 + 납입완료 │ 초록색 (안심)   │
├────────────────────────────────────┤
│ 하원 + 납입완료   │ 연한 빨강 (양호)│
├────────────────────────────────────┤
│ 등원중 + 미납     │ 노란색 (주의)   │
├────────────────────────────────────┤
│ 하원 + 미납       │ 진한 빨강 (경고)│
└────────────────────────────────────┘


📊 통계 활용
===========

엑셀 파일에서 필터링:

1. D열 필터 적용
2. 빈칸 선택 → 미납 학생만 표시
3. 미납 학생 명단 확인
4. 개별 연락 및 독촉


🔧 문제 해결
===========

Q: 납입등록 버튼이 안 보여요
A:
  - GUI 프로그램 재시작
  - 최신 버전인지 확인
  - config.json에 "payment_column": "D" 있는지 확인

Q: 날짜를 잘못 입력했어요
A:
  - [납입등록] 버튼 다시 클릭
  - 올바른 날짜로 재입력
  - 덮어쓰기 됩니다

Q: 납입 정보를 삭제하고 싶어요
A:
  - [납입등록] 버튼 클릭
  - 빈칸으로 확인
  - 또는 엑셀에서 D열 셀 삭제

Q: 여러 학생을 한번에 등록하고 싶어요
A:
  - 엑셀 파일에서 직접 수정
  - D열에 날짜 일괄 입력
  - 복사/붙여넣기 활용
