========================================
등원/하원 입력 프로그램 사용법
========================================

엑셀 파일을 직접 수정하지 않고 프로그램에서 
간편하게 등원/하원 처리를 할 수 있습니다!

📌 실행 방법
-----------

방법 1: 등하원입력.bat 실행 (선택 가능)
  → 콘솔 버전 또는 GUI 버전 선택

방법 2: 등하원입력_콘솔.bat 실행
  → 텍스트 기반 콘솔 프로그램

방법 3: 등하원입력_GUI.bat 실행 (추천!)
  → 그래픽 인터페이스 프로그램


🖥️ GUI 버전 (그래픽 인터페이스) - 추천!
------------------------------------

[등하원입력_GUI.bat] 실행

특징:
✓ 예쁜 그래픽 화면
✓ 마우스 클릭으로 간편 조작
✓ 자동 새로고침 (3초마다)
✓ 색상으로 상태 구분
  - 초록색: 등원중
  - 빨간색: 하원 상태

사용 방법:
1. 학생 이름을 클릭하여 선택
2. [등원] 또는 [하원] 버튼 클릭
3. 완료 메시지 확인
4. 자동으로 문자 발송됨!


📟 콘솔 버전 (텍스트 기반)
--------------------------

[등하원입력_콘솔.bat] 실행

특징:
✓ 빠른 실행
✓ 키보드로 조작
✓ 간단명료

사용 방법:
1. 학생 목록이 번호와 함께 표시됨
2. 학생 번호 입력
3. 1(등원) 또는 2(하원) 선택
4. Enter 키로 확인
5. 자동으로 문자 발송됨!


💡 사용 예시
----------

[상황 1] 홍길동 학생이 학원에 도착
  → 등하원입력_GUI.bat 실행
  → 홍길동 선택
  → [등원] 버튼 클릭
  → "홍길동님이 OO학원에 등원하였습니다" 문자 자동 발송

[상황 2] 김영희 학생이 학원에서 퇴원
  → 등하원입력_GUI.bat 실행
  → 김영희 선택
  → [하원] 버튼 클릭
  → "김영희님이 OO학원에서 하원하였습니다" 문자 자동 발송


🔄 자동 알림 연동
---------------

이 입력 프로그램과 자동 알림 프로그램을 함께 사용하세요!

1. [실행.bat] 실행 → 자동 알림 모니터링 시작
   (백그라운드에서 계속 실행)

2. [등하원입력_GUI.bat] 실행 → 등/하원 입력
   (필요할 때마다 실행)

3. 입력 프로그램에서 상태 변경
   → 엑셀 파일 자동 업데이트
   → 모니터링 프로그램이 변화 감지
   → 자동으로 문자 발송!


⚙️ 설정
------

config.json 파일에서 설정 변경 가능:
- academy_name: 학원 이름
- check_interval: 체크 주기
- 컬럼 설정 등


📝 주의사항
---------

1. 엑셀 파일이 열려있으면 저장이 안될 수 있습니다
   → 엑셀을 닫고 사용하세요

2. 이미 등원중인 학생을 다시 등원 처리하면
   → 경고 메시지가 표시됩니다

3. 하원 상태인 학생을 다시 하원 처리하면
   → 경고 메시지가 표시됩니다


🎯 추천 사용 방법
---------------

[방법 A] GUI만 사용 (간편)
  등하원입력_GUI.bat만 실행하고
  이 프로그램에서 모든 처리
  (자동 알림은 백그라운드에서 따로 실행)

[방법 B] 콘솔 버전 사용 (빠름)
  키보드만으로 빠르게 처리
  자주 사용하는 경우 추천


문의사항이 있으면 README.md 파일을 참조하세요!
